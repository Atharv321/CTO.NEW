# Shared Package

Shared utilities, types, and constants used across the monorepo.

## Usage

### Types

```typescript
import type { User } from '@shared/shared/types';
```

### Utils

```typescript
import { formatDate, parseJSON } from '@shared/shared/utils';
```

## Development

```bash
# Build
pnpm run build

# Tests
pnpm run test

# Type checking
pnpm run type-check
```
